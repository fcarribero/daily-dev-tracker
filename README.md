## Daily Dev Tracker

Aplicación Laravel + Blade sin autenticación para registrar entradas de trabajo diarias y generar un texto de "daily" en base a un día.

Características:
- Crear múltiples entradas por día, en texto plano.
- Listar entradas por fecha.
- Botón para generar "Daily" (lista concatenada de entradas del día).
- UI con Bootstrap 5.
- Base de datos SQLite local.
 - Generación de resumen de Daily con IA (ChatGPT) o fallback local.

### Requisitos
- Docker y Docker Compose instalados.

### Cómo ejecutar
1. Construir e iniciar el contenedor:
   ```bash
   docker compose up --build
   ```
2. Abrir en el navegador:
   - http://localhost:8080

Notas:
- El contenedor crea automáticamente `database/database.sqlite` si no existe, genera `APP_KEY` y corre migraciones al arrancar.
- La configuración de la BD usa SQLite (`.env` ya configurado para Docker).
 - Para usar la generación con IA, define tu `OPENAI_API_KEY` en `.env` dentro del contenedor o del proyecto.

### Estructura básica
- Rutas en `routes/web.php`.
- Modelo `App\Models\Entry` y migración de `entries`.
- Controlador `App\Http\Controllers\EntryController`.
- Vistas en `resources/views/entries` y layout en `resources/views/layouts/app.blade.php`.
 - Servicio de IA: `App\Services\DailySummaryGenerator`.

### Configurar ChatGPT (OpenAI)
1. Obtén una API Key de OpenAI y colócala en `.env`:
   ```env
   OPENAI_ENABLED=true
   OPENAI_API_KEY=sk-...
   OPENAI_MODEL=gpt-4o-mini
   # OPENAI_BASE_URL=https://api.openai.com/v1  # opcional
   # OPENAI_TIMEOUT=20                          # opcional
   ```
2. Reinicia el contenedor si estás usando Docker:
   ```bash
   docker compose restart
   ```
3. Si `OPENAI_API_KEY` no está configurada o hay un error con la API, la app usará un resumen local simple como fallback.
